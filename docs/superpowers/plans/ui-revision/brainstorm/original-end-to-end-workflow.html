Design 2 of 5 · Upload-to-download workflow

One guided pipeline, split into an analysis phase and a user-triggered render phase.

1

Create project

Direct multipart upload to MinIO, then probe the source.

  • MP4, MOV, MKV, WebM
  • Up to 3 hours / 10 GB
  • Resumable upload
2

Configure analysis

Review a conservative preflight estimate before spending.

  • Language
  • Model + reasoning
  • USD cap + 1.5× reserve
  • Max clips + max seconds
  • Optional creative instruction
3

Analyze locally

Temporal coordinates resumable stages on the native Mac worker.

  • Extract audio + proxy
  • Local transcription
  • Verified budget gate
  • OpenAI transcript ranking
  • Face tracking + previews
4

Review and edit

Filmstrip, vertical preview, inspector, and trim timeline.

  • Accept / reject candidates
  • Adjust boundaries + focal point
  • Edit caption text + styling
  • Choose platform safe-area preset
5

Render and download

Render selected clips only, with audio and captions stitched.

  • 1080 × 1920 MP4
  • H.264 + AAC
  • Individual downloads
  • Download-all archive
